# AEGIS — Autonomous Enterprise & Guardian Intelligence System

AEGIS is a production-grade AI-native demo platform that simulates a next-generation security operating system. Built with React, Tailwind CSS, and Framer Motion, it visualizes complex multi-agent orchestration and threat prediction workflows.

## 🚀 Live Demo Presentation

### 1. The Landing Page (The Pitch)
- **Hero Hook**: High-impact headlines with animated gradients and a real-time GitHub star count.
- **Bento Grid**: Premium cards detailing the "AI Brain," "Autonomous Agents," and "Graph Intelligence."
- **Vision & Roadmap**: A concise, professional breakdown of the system's purpose and future phases.

### 2. The OS Console (The Core Experience)
- **Boot Sequence**: Immersive initialization sequence showing kernel readiness and agent handshakes.
- **System Aliveness**: Background "pulses" and idle logs ensure the system feels active within 5 seconds.
- **Reality Seed Flow**: Users select from 3 distinct scenarios (Ransomware, Login Anomaly, Data Leak) to trigger the intelligence parsing engine.
- **Dynamic Graph (SSOT)**: A state-driven central visualization that acts as the single source of truth, showing nodes (Servers, Users, APIs) and threat trajectories.
- **Unified Command Feed**: A structured, agent-tagged log feed ([HUNT], [INTEL], [RED], [BLUE]) with terminal-style block cursor animations.

### 3. Intelligence Report (The Decision Tool)
- **Fidelity Metrics**: Data-driven summaries showing Alert Noise Reduction and Response Velocity.
- **Scenario Insights**: Tailored logic for each threat type, providing actionable intelligence.
- **Autonomous Actions**: A log of decisions made by the agent swarm to neutralize threats.

## 🛠️ Tech Stack

- **Frontend**: React + TypeScript + Vite
- **Styling**: Tailwind CSS (Custom Design System)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Logic**: Mock Simulation Engine with state-driven state transitions.

## 🎨 Design System

- **Dark Mode (Default)**: Background: `#0B0F14`, Cards: `#111827`, Accent: `#4F9CF9`.
- **Light Mode**: Premium corporate white with soft gradients and high-contrast typography.
- **Typography**: Modern Sans-serif for UI, JetBrains Mono for system logs.

## 📂 Project Structure

- `src/pages`: Landing and Console page components.
- `src/components`: Reusable UI elements including the Dynamic Graph.
- `src/context`: Theme and state management.
- `src/utils`: Simulation logic and scenario-specific data.
- `src/types`: Centralized TypeScript definitions.
